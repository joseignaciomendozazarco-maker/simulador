# Simulador de Expedición Bidimensional - Marte

[cite_start]Este proyecto es una aplicación interactiva desarrollada en Python 3.11 que simula los movimientos y recolección de datos de un vehículo explorador (Rover) en un mapa bidimensional limitado de -100 a 100 unidades en los ejes X e Y[cite: 8, 49].

## Requisitos de Ejecución
* [cite_start]Disponer de **Python 3.11** instalado en el sistema[cite: 106].
* [cite_start]No requiere la instalación de ninguna librería de terceros (`pip`), utiliza únicamente módulos de la biblioteca estándar (`math`, `random` y `turtle`)[cite: 112, 113, 117].

## Instrucciones de Uso
1. Ejecuta la consola de comandos de tu sistema operativo dentro de la carpeta raíz del proyecto.
2. Introduce el siguiente comando para iniciar el simulador:
   ```bash
   python main.py
